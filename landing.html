perfkit

Capture, store, compare Go profiles

A minimal pprof collector with a clean web interface. Zero dependencies beyond SQLite.

go install github.com/flaticols/perfkit/cmd/perfkit@latest

Features

01

Capture from any Go app

Connect to any application with pprof enabled. CPU, heap, goroutine, mutex, block profiles.

02

Web UI for browsing

Clean interface to browse profiles by session, project, or type. Dark mode included.

03

Compare with deltas

Select multiple profiles and see metric changes over time. Track regressions instantly.

04

Session grouping

Organize related profiles under sessions. Perfect for load tests or debugging sessions.

05

SQLite storage

Single file database. No external services. Just run the binary.

Quick start

Start the server and begin capturing profiles in under a minute.

# Start the server
perfkit server

# Capture profiles from your app
perfkit capture http://localhost:6060 --session load-test

# Or capture periodically
perfkit capture http://localhost:6060 --interval 30s

Open http://localhost:8080 to view profiles.
